Tutto quello che ti serve

Da homelab a infrastruttura aziendale — NovaSCM si adatta alle tue esigenze.

📡

Scansione rete avanzata

Scopre automaticamente tutti i device — IP, MAC, vendor, tipo, porte. Vista tabella, mappa hub-and-spoke e heatmap subnet.

🔐

WiFi EAP-TLS enterprise

Genera e distribuisce certificati per autenticazione WiFi WPA2-Enterprise. Enrollment automatico su PC Windows con un comando.

💿

Deploy Windows zero-touch

Genera autounattend.xml + postinstall.ps1. Installa Windows 11 senza premere un tasto — USB o PXE.

⚙️

Workflow automation

Sequenze di step automatizzati eseguiti sui PC remoti. Simile alle Task Sequence SCCM ma senza l'infrastruttura SCCM.

🖥️

Fleet PC management

Inventario hardware/software via WMI, RDP one-click, gauge risorse in tempo reale (CPU/RAM/disco/rete).

📋

Change Requests

Traccia il ciclo di vita dei PC dal provisioning alla consegna. Funziona offline con sync automatica.

📖

Wiki integrata

Documentazione completa direttamente nell'app — nessun browser necessario per trovare le istruzioni.

🏢

Integrazione SCCM

Visualizzatore Task Sequence, monitoraggio OSD, compatibile con Microsoft Endpoint Configuration Manager.

Inizia subito

Pronto in 60 secondi. Nessuna configurazione obbligatoria.

# Scarica e avvia — richiede .NET 9.0
git clone https://github.com/claudiobecchis/NovaSCM
cd NovaSCM && dotnet run

Oppure scarica direttamente l'eseguibile precompilato →
